# Tabnine

Tabnine is built on a **zero data retention** promise — your code is never stored, never used for training, and never shared. For teams in regulated industries (healthcare, finance, government), this makes Tabnine the go-to AI assistant. **Personalized completions** learn from your team's coding patterns without retaining the data itself. MCP tools add external context under the same privacy guarantee.

- **Agent mode** — autonomous coding with MCP tools

## Setup

### 1. Create a Token
In [Vinkius Cloud](https://cloud.vinkius.com), go to your server → **Connection Tokens** → **Create**. Copy the URL.

### 2. Configure MCP
In Tabnine → **Settings** → **MCP** → add your URL.

### 3. Private and Connected
AI completions stay private. MCP tools provide controlled external access.
