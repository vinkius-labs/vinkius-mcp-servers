# ChatWise

ChatWise brings AI-assisted development to Apple's ecosystem with **iCloud sync** across Mac, iPhone, and iPad. Set up MCP on your Mac, and the same tools and conversation history are available on your phone. For developers, this means reviewing MCP-powered code analysis on your commute or checking deployment status from your phone using the same AI configuration.

## Cross-Device AI Development for Apple Users

Features:
- **iCloud sync** — conversations and MCP configs across Apple devices
- **Siri Shortcuts** — voice-activated MCP queries
- **SwiftUI** — native Apple design with haptics and Dark Mode
- **Code highlighting** — syntax-aware markdown rendering
- **Multi-provider** — OpenAI, Anthropic, and other providers
- **Privacy** — data stored in your iCloud, not third-party servers

## Setup

### 1. Create a Token
In [Vinkius Cloud](https://cloud.vinkius.com), go to your server → **Connection Tokens** → **Create**. Copy the URL.

### 2. Add MCP
In ChatWise → **Settings** → **MCP** → add your URL. iCloud syncs to all devices.

### 3. Any Device
Code on Mac. Review on iPhone. Same MCP tools everywhere.
