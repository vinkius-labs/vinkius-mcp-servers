## Menu Bar AI with Instant MCP Access

Features:
- **Menu bar** — always accessible via keyboard shortcut
- **Scratchpad** — persistent workspace for extended AI sessions
- **Prompt templates** — reusable templates for common workflows
- **Floating window** — AI overlay on top of any app
- **Multi-provider** — OpenAI, Anthropic, Google, Ollama
- **Image generation** — DALL-E and Stable Diffusion
- **macOS native** — Swift, Apple Silicon optimized

## Setup

### 1. Create a Token
In [Vinkius Cloud](https://cloud.vinkius.com), go to your server → **Connection Tokens** → **Create**. Copy the URL.

### 2. Add MCP
In Witsy → **Preferences** → **MCP** → add your URL.

### 3. Quick Access
Press your shortcut from any app. MCP tools available instantly.
