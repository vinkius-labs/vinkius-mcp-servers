# MCPJam

MCPJam is a community-driven platform for discovering MCP servers, tools, and resources. It curates a directory of MCP server implementations, client compatibility information, tutorials, and community projects. Use MCPJam to find servers that complement your Vinkius Cloud setup, or list your own tools for the community.

## The Community Hub for MCP

MCPJam aggregates the MCP ecosystem in one place. Browse server implementations by category — databases, APIs, monitoring, development tools — read community reviews, and find client compatibility information. It also hosts tutorials for building custom MCP servers.

Use MCPJam alongside Vinkius Cloud to discover additional MCP servers that complement your setup. Or submit your own Vinkius-powered servers to the directory.

What MCPJam provides:

- **Server directory** — curated list of MCP server implementations
- **Client compatibility** — which clients work with which servers
- **Tutorials** — guides for building and deploying MCP servers
- **Community** — discussions, project showcases, and resource sharing
- **Search** — filter servers by language, category, and transport type

## Getting Started

### 1. Browse MCPJam

Visit [mcpjam.com](https://mcpjam.com) to explore available MCP servers and resources.

### 2. Create a Vinkius Token

In [Vinkius Cloud](https://cloud.vinkius.com), go to your server → **Connection Tokens** → **Create**. Copy the URL.

### 3. Connect with Any Client

Use any MCP client to connect. MCPJam's compatibility guides help you choose the right client for your use case.

## FAQ

**Is MCPJam an MCP client?**
MCPJam is a community directory, not a client. It helps you discover servers and choose compatible clients.

**Can I list my server on MCPJam?**
Yes. Submit your MCP server implementation to the community directory.

**Is MCPJam free?**
Yes. The community platform is free to use.

**How does MCPJam relate to Vinkius?**
They are complementary. Vinkius Cloud hosts your MCP servers. MCPJam helps you discover additional servers and resources.
